# Канонический шаблон README с LLM-мета-секцией

**Приоритет:** P1 · **Зависимости:** нет (описания согласовать с packages-inventory.md) · **Трекер:** DATAUI-3745, EPIC 2

## Цель

Единый шаблон README, удобный людям и парсабельный генераторами. README — источник истины: из него собираются пакетные llms.txt, из его мета-секции — корневой каталог. Это «шаблон, который всем репам надо будет у себя реализовать» из саммари встречи 25.06.

## Шаги

1. Спроектировать шаблон README пакета. Обязательные блоки (маркированы HTML-комментариями для машинного парсинга, например `<!-- LLM_META_START/END -->`):
   - **Positioning** — что делает пакет, одно предложение (= `llmDescription` из libs.mjs; единый источник — решить где: предлагаю libs.mjs генерит секцию в README через PR-бот, или наоборот README — источник, а trip-wire сверяет);
   - **When to use / When not to use** — паттерны и антипаттерны, явные отсылки к соседям («для дашбордных графиков берите charts, chartkit — legacy-обёртка»);
   - **Required setup** — импорты стилей, обёртки, порядок инициализации;
   - **Peer dependencies** — таблица с минимальными версиями, включая peer-граф внутри @gravity-ui/\*;
   - **Common pitfalls / Hallucination traps** — имена, которые модели выдумывают (несуществующие пропсы/компоненты), с указанием правильных (идея Камендровского из комментариев к DATAUI-3745);
   - Install / Usage / ссылки на доки сайта и Storybook — как сейчас.
2. Шаблон README компонента (для uikit и других компонентных пакетов): описание одним предложением первой строкой (её берут генераторы), пропсы таблицей, минимальный рабочий пример, без вербозных развёрнутых примеров «для дизайнера».
3. Правила чистоты: бейджи и картинки — только в шапке до мета-секции; сервисные маркеры (`<!--SANDBOX-->`, `<!--GITHUB_BLOCK-->`) не внутри мета-секции.
4. Валидатор шаблона — скрипт, проверяющий наличие/формат блоков; подключить в CI реп через общий reusable workflow. Trip-wire из packages-inventory.md ссылается на него.
5. Пилот: переписать README `@gravity-ui/uikit` (пакетный) + 3–5 README компонентов по шаблону, показать команде, утвердить.
6. План раскатки по тирам (Tier-1 → Tier-2 → Tier-3) с мотивацией мейнтейнеров — вики-страница и рассылка (у ngorin по саммари 25.06).

## Definition of Done

- [ ] Шаблоны (пакетный + компонентный) утверждены и опубликованы (вики + CONTRIBUTING).
- [ ] Валидатор работает в CI хотя бы одного репозитория.
- [ ] README uikit приведён к шаблону (пилот).
- [ ] Согласован единый источник описаний (README ↔ libs.mjs) — без дублирования, с проверкой рассинхрона в CI.
