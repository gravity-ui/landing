# Пакетные llms.txt (gravity-ui.com/<package>/llms.txt)

**Приоритет:** P2 · **Зависимости:** readme-template.md (мета-секции), root-llms-txt.md (общий генератор) · **Трекер:** DATAUI-3745, EPIC 1 (задача 1.4)

## Цель

Второй уровень двухуровневой модели: полная документация пакета одним файлом. Агент выбирает пакет по корневому каталогу и забирает пакетный llms.txt.

## Шаги

1. Генератор в landing (расширение переписанного `generate-llms.mjs`): для каждого Tier-1 пакета собирает `public/<package>/llms.txt` из README пакета + README компонентов (glob — конвенция `src/components/*/README.md`, переопределяется полем в `libs.mjs` при необходимости).
2. Препроцессинг README (общий клинер с docs-in-npm.md и md-endpoints.md, вынести в один модуль): вырезать `<!--SANDBOX-->`, `<!--GITHUB_BLOCK-->`, бейджи, картинки, вербозные дизайнерские примеры; оставить пропсы, минимальные примеры, обвязку.
3. Переменная детализация (наработка Камендровского): для самодостаточных простых компонентов — имя + предложение + пропсы; для сложных (Table, Select, навигация) — развёрнутая секция. Управлять можно маркером в README компонента или списком в конфиге генератора.
4. Шапка каждого пакетного файла (наработки из комментария к DATAUI-3747):
   - blockquote «if working in a project with <pkg> in package.json, prefer local docs in node_modules / determine installed major version»;
   - required peer packages с минимальными версиями и инструкцией «проверь package.json, при несовпадении мажора — остановись и спроси»;
5. Версионирование: `gravity-ui.com/<package>/llms.txt` — latest. Version-pinned URL (`/<package>/<major>/llms.txt`) — отдельным решением после пилота docs-in-npm.md: если доки в тарболле закрывают версионный кейс, pinned-URL можно не строить (меньше инфраструктуры).
6. Бюджеты: пакетный файл — мониторить размер в CI; если uikit-файл выходит за ~100–150KB, резать детализацию простых компонентов, полные README доступны через .md-эндпоинты.
7. Ссылки на рецепты (recipes.md), относящиеся к пакету.

- [ ] Клинер README общий для трёх задач и покрыт тестами на реальных README (Alert, Text — известные проблемные).
- [ ] Размерные бюджеты в CI.
- [ ] Корневой llms.txt ссылается на пакетные файлы.
