# 骷髅

<!--/GITHUB_BLOCK-->

```tsx
import {Skeleton} from '@gravity-ui/uikit';
```

Skeleton 组件会在数据加载之前显示内容的占位符预览。显示此预览是为了减少加载时间的麻烦。

## 属性

| 姓名      | 描述                          |         类型          | 默认 |
| :-------- | :---------------------------- | :-------------------: | :--: |
| 风格      | 根元素的自定义 CSS 属性       | `React.CSSProperties` |      |
| className | 根元素的自定义 CSS 类         |       `string`        |      |
| qa        | `data-qa` HTML 属性，用于测试 |       `string`        |      |
