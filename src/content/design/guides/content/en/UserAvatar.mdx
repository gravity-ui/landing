A user avatar is used to quickly identify a specific user or account and to display the authorized user.

# Size

![1.png](/static/images/design/UserAvatar/1.png)

**XL** - 50 x 50 px

**L** - 42 x 42 px

**M** - 32 x 32 px (Default)

**S** - 28 x 28 px

**XS** - 24 x 24 px

# Types of content

![2.png](/static/images/design/UserAvatar/2.png)

**No images (default):** If the image source is missing or a loading error occurs, the default placeholder will be used to fill the container.

**Uploaded image:** If a user uploads an image, that image will be used as a placeholder for the entire avatar area. Images fill the entire container width (100%) by default.

# Functionality


**Actions on click or hover:** Optionally, user avatars may contain additional information that is displayed upon a click or hover.

![2.png](/static/images/design/UserAvatar/2%201.png)

**Custom size values:** Optionally, the size of the avatar can be set to a custom value.Use the default size when possible.
