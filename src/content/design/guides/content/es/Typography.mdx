La tipografía se basa en el uso de dos fuentes y un conjunto de estilos que incluyen fuente, grosor, tamaño de fuente y interlineado.

De forma predeterminada, se utilizan dos fuentes: Inter para la composición tipográfica estándar y los encabezados, y Menlo como fuente monoespaciada. Inter está disponible en [Google Fonts](https://fonts.google.com/specimen/Inter?query=inter) y Menlo viene preinstalado en algunos sistemas operativos. Puedes cambiar estas fuentes de acuerdo con las directrices de tu marca.

# Parámetros de fuente

### Talla y entallado

El interlineado determina el espacio entre las líneas base y el tamaño de fuente establece el tamaño de la fuente. Estos dos parámetros determinan la legibilidad del texto.

### Peso

El peso se especifica para los estilos de texto (el valor predeterminado para los grupos Cuerpo y Título es 400) y los estilos de visualización (el valor predeterminado para el subencabezado, el encabezado y la visualización es 600). De forma predeterminada, los estilos de visualización tienen más peso. Sin embargo, puede haber excepciones.

# Grupos de estilos

### **Cuerpo**

Se usa para todos los tipos de texto. El valor predeterminado es Inter (400).

![Body](/static/images/design/Typography/Body.png)

Body 1 Short ha reducido el interlineado y se usa en bloques de contenido estrechos (de hasta 300 a 400 píxeles de ancho).

### **Subtítulo**

Las fuentes más pequeñas de todas las fuentes disponibles. Se utilizan como estilos para subtítulos, notas a pie de página y cualquier información adicional. El valor predeterminado es Inter (400).

![Caption](/static/images/design/Typography/Caption.png)

### **Cabecera**

Se utilizan con mayor frecuencia como encabezados en interfaces donde la densidad de información es crucial. El valor predeterminado es Inter (600).

![Header](/static/images/design/Typography/Header.png)

### **Subtítulo**

Se puede usar como subtítulos o como acentos cuando se combina con los estilos de cuerpo apropiados. El valor predeterminado es Inter (600).

![Subheader](/static/images/design/Typography/Subheader.png)

### **Pantalla**

Adecuado para situaciones en las que la visualización es más importante que la densidad de la información, como en el diseño de la comunicación. El valor predeterminado es Inter (600).

![Display](/static/images/design/Typography/Display.png)

### **Código**

Se usa para mostrar el código en bloques separados. Menlo está configurado como predeterminado.

![Code](/static/images/design/Typography/Code.png)

### **Código en línea**

Se usa para generar código incrustado en texto plano. El espaciado entre líneas reducido se utiliza para la compensación óptica y la alineación con texto plano. Al cambiar las fuentes, asegúrese de que el código en línea se muestre correctamente. Menlo está configurado como predeterminado.

![Code Inline](/static/images/design/Typography/CodeInline.png)
