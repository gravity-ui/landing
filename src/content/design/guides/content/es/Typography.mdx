La tipografía se basa en el uso de dos fuentes y un conjunto de estilos que incluyen tipo de letra, peso, tamaño de fuente y interlineado.

Por defecto se utilizan dos fuentes: Inter para la composición tipográfica estándar y títulos, y Menlo como fuente monoespaciada. Inter está disponible en [Google Fonts](https://fonts.google.com/specimen/Inter?query=inter), y Menlo viene preinstalada en algunos sistemas operativos. Puede cambiar estas fuentes de acuerdo con las directrices de su marca.

# Parámetros de fuente

### Tamaño e interlineado

El interlineado determina el espacio entre las líneas base, y el tamaño de la fuente establece el tamaño de la letra. Estos dos parámetros determinan la legibilidad del texto.

### Peso

El peso se especifica para estilos de texto (el valor predeterminado para los grupos Body y Caption es 400) y estilos de visualización (el valor predeterminado para Subheader, Header y Display es 600). Por defecto, los estilos de visualización tienen más peso. Sin embargo, puede haber excepciones.

# Grupos de estilos

### **Body**

Se utiliza para todo tipo de texto. Inter se establece como predeterminado (400).

![Body](/static/images/design/Typography/Body.png)

Body 1 Short tiene un interlineado reducido y se utiliza dentro de bloques de contenido estrechos (hasta 300-400px de ancho).

### **Caption**

Las fuentes más pequeñas de todas las disponibles. Se utilizan como estilos para leyendas, notas al pie y cualquier información adicional. Inter se establece como predeterminado (400).

![Caption](/static/images/design/Typography/Caption.png)

### **Header**

Se utiliza más comúnmente como encabezados en interfaces donde la densidad de información es crucial. Inter se establece como predeterminado (600).

![Header](/static/images/design/Typography/Header.png)

### **Subheader**

Puede utilizarse como subtítulos o como acentos cuando se combina con estilos de cuerpo apropiados. Inter se establece como predeterminado (600).

![Subheader](/static/images/design/Typography/Subheader.png)

### **Display**

Adecuado para situaciones donde la visualización es más importante que la densidad de información, como en el diseño de comunicación. Inter se establece como predeterminado (600).

![Display](/static/images/design/Typography/Display.png)

### **Code**

Se utiliza para mostrar código en bloques separados. Menlo se establece como predeterminado.

![Code](/static/images/design/Typography/Code.png)

### **Code Inline**

Se utiliza para mostrar código que está incrustado en texto plano. Se utiliza un espaciado de línea reducido para compensación óptica y alineación con texto plano. Al cambiar las fuentes, asegúrese de que el código en línea se muestre correctamente. Menlo se establece como predeterminado.

![Code Inline](/static/images/design/Typography/CodeInline.png)
