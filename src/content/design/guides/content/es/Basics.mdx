Gravity-UI es una colección de bibliotecas y componentes para crear interfaces de usuario funcionales. El sistema de diseño admite varios temas: **variaciones** claras ****, **oscuras y** de alto contraste de cada uno. El estilo visual de los componentes de cada tema se puede personalizar fácilmente.

**Color**

El sistema de diseño consiste en cuatro temas basados en un modelo de color de dos capas. El motor de temas admite una personalización versátil del color y las identidades de marca con diferentes conjuntos de requisitos.

**Tipografía**

El sistema tipográfico consiste en estilos de composición tipográfica, fuentes de visualización y código (fuente monoespacial). Puedes usar las fuentes predeterminadas o crear las tuyas propias.

**Radio de esquina de control**

El radio de las esquinas es una parte pequeña pero importante del estilo visual. Para que sea más fácil trabajar con él, hemos desarrollado un conjunto de variables de radio. Estas variables son utilizadas por diferentes controles. Para ajustar el radio de los controles, basta con cambiar los valores de las variables.

**Creación y uso de un tema**

Sea cual sea la marca y el lenguaje visual, todo se reduce a un conjunto de constantes que deberías poder editar y usar fácilmente tanto en tu código como en la biblioteca de Figma.
