Gravity-UI es una colección de bibliotecas y componentes para construir interfaces de usuario funcionales. El sistema de diseño soporta múltiples temas: variaciones **clara**, **oscura** y de **alto contraste** de cada una. El estilo visual de los componentes en cada tema se puede personalizar fácilmente.

**Color**

El sistema de diseño consta de cuatro temas basados en un modelo de color de dos capas. El motor de temas soporta una personalización versátil del color y la identidad de marca con diferentes conjuntos de requisitos.

**Tipografía**

El sistema de tipografía consta de estilos de composición, fuentes de visualización y código (fuente monoespaciada). Puedes usar las fuentes predeterminadas o crear las tuyas propias.

**Controlar el radio de las esquinas**

El radio de las esquinas es una parte pequeña pero importante del estilo visual. Para facilitar su manejo, hemos desarrollado un conjunto de variables de radio. Estas variables son utilizadas por diferentes controles. Para ajustar el radio de los controles, simplemente cambia los valores de las variables.

**Crear y usar un tema**

Independientemente de la marca y el lenguaje visual, todo se reduce a un conjunto de constantes que deberías poder editar y usar fácilmente tanto en tu código como en la biblioteca de Figma.
