Gravity-UI es una colección de bibliotecas y componentes para construir interfaces de usuario funcionales. El sistema de diseño admite múltiples temas: variaciones **claras**, **oscuras** y de **alto contraste** de cada uno. El estilo visual de los componentes en cada tema puede personalizarse fácilmente.

**Color**

El sistema de diseño consta de cuatro temas basados en un modelo de color de dos capas. El motor de temas admite personalización versátil de colores e identidades de marca con diferentes conjuntos de requisitos.

**Tipografía**

El sistema de tipografía consta de estilos de composición tipográfica, fuentes de visualización y código (fuente monoespaciada). Puede usar las fuentes predeterminadas o crear las suyas propias.

**Radio de esquina de control**

El radio de esquina es una parte pequeña pero importante del estilo visual. Para facilitar el trabajo con él, desarrollamos un conjunto de variables de radio. Estas variables son utilizadas por diferentes controles. Para ajustar el radio de los controles, simplemente cambie los valores de las variables.

**Creación y uso de un tema**

Independientemente de la marca y el lenguaje visual, todo se reduce a un conjunto de constantes que debería poder editar y usar fácilmente tanto en su código como en la biblioteca de Figma.
