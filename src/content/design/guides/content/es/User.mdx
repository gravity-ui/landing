Un componente para mostrar un avatar junto con el nombre de usuario y la información de contacto, con contenido que se puede modificar fácilmente.

# Estructura

![Specs3.png](/static/images/design/User/Specs3.png)

1. Avatar de usuario (componente Avatar)
2. Bloqueo de nombre de usuario (opcional)
3. Bloque de contactos (opcional)

# Tamaño

![7.png](/static/images/design/User/7.png)

**XL**: avatar de 50x50 px y marco de texto que se puede ajustar al ancho del contenido

**L** - Avatar de 42x42 px + marco de texto que se puede ajustar al ancho del contenido

**M**: avatar de 32x32px (predeterminado) + marco de texto que se puede ajustar al ancho del contenido

**S**: avatar de 28x28 px y marco de texto que se puede ajustar al ancho del contenido

**XS**: avatar de 24 x 24 píxeles y marco de texto que se puede ajustar al ancho del contenido

# Tipos de contenido

![11.png](/static/images/design/User/11.png)

**Bloque completo (predeterminado)**

De forma predeterminada, el componente está configurado para mostrar la cantidad máxima de contenido, incluidos un avatar, una sección de nombres y una sección de contactos.

**Mostrar sin avatar de usuario**

Opcionalmente, se puede desactivar el bloque de avatar del usuario.

# Funcionalidad

![10.png](/static/images/design/User/10.png)

**Configuración flexible de una pantalla opcional**

Si es necesario, se puede desactivar cualquier bloque de componentes.
