## Descripción

Con el componente Tabs, puede organizar el contenido dividiéndolo en varias secciones entre las que el usuario puede alternar para visualizar. Use pestañas para agrupar secciones relacionadas y proporcionar acceso rápido a cada una de ellas.

![1.png](/static/images/design/Tabs/tabs-0001.png)

## Tamaños

Las pestañas pueden tener tres tamaños:

![2.png](/static/images/design/Tabs/tabs-0002.png)

- **M (Medio):** Use este tamaño para uso general en páginas, dentro de secciones individuales.
- **L (Grande):** Use el tamaño L si las pestañas también sirven como subtítulos de página. En este caso, el contenido de la pestaña es una sección separada en la página, y las pestañas son subtítulos de esta sección.
- **XL (Extra Grande)**: Este tamaño es apropiado cuando las pestañas también sirven como encabezados de página. En este caso, cuando los usuarios cambian de pestaña, el contenido de toda la página cambia.

## Estados

Las pestañas pueden tener los siguientes estados:

![3.png](/static/images/design/Tabs/tabs-0003.png)

- **Inactivo**: La pestaña está en un estado inactivo cuando no está seleccionada. En este estado, la pestaña tiene un color más desvanecido para resaltar la pestaña activa.
- **Activo**: Este estado se asigna a la pestaña seleccionada que muestra la sección activa actual del contenido. Las pestañas activas tienen colores más saturados y están subrayadas para enfatizar.
- **Hover**: Cuando el usuario pasa el cursor sobre la pestaña, el puntero cambia su estilo para indicar la posibilidad de interacción.
- **Deshabilitado**: Las pestañas pueden estar deshabilitadas si están temporal o permanentemente no disponibles para selección. Tales pestañas están en gris y están inactivas para la interacción.

## Propiedades

Puede habilitar/deshabilitar los siguientes elementos para cada pestaña:

![4.png](/static/images/design/Tabs/tabs-0004.png)

1. Icono. Se utiliza a discreción del diseñador para visualizar el tipo de contenido dentro de la pestaña.
2. Etiqueta. El nombre de la sección en formato de texto.
3. Contador. Puede usarlo para indicar el número de entidades dentro de una pestaña. También puede usarlo para especificar el número de nuevas entidades que surgieron en la pestaña.
4. Etiqueta. Puede usarla para denotar el estado de la sección de la que la pestaña es responsable.

## Recomendaciones de uso

Use pestañas para organizar grandes volúmenes de contenido o para dividir la información en secciones lógicas. Asegure una estructura clara y fácilmente legible para los encabezados de las pestañas.

Seleccione el estilo y los colores de las pestañas que coincidan con el diseño general de su proyecto. Siempre resalte las pestañas activas para que los usuarios puedan identificar fácilmente la sección actual.

Siempre que sea posible, intente que sus secciones de pestañas M y L quepan en una sola altura de pantalla. De esta manera, puede asociar fácilmente el contenido con el nombre de la pestaña, y ver dónde termina el contenido de la pestaña y comienza el siguiente bloque.
