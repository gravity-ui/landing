El botón de copiar al portapapeles es una modificación del componente `Button` `Flat` con el icono `ClipboardIcon` colocado por separado.

### Icono

Dentro del `ClipboardButton` se encuentra un componente `ClipboardIcon`, que es un icono animado con dos estados: Éxito y Error. Para personalizar el botón de copia, utiliza el componente de botón estándar `Button` con el icono `ClipboardIcon`.

### Estados

![Estados](/static/images/design/ClipboardButton/States.png)

### Tamaños

El componente soporta los tamaños de botón estándar `XS`, `S`, `M`, `L` y `XL`.

### Comportamiento

Si la copia de contenido no se realiza con frecuencia o si el icono reduce la legibilidad, el botón de copia no aparecerá hasta que el cursor se acerque.

![En tablas, el icono se oculta hasta que el cursor se coloca sobre una celda con contenido](/static/images/design/ClipboardButton/Use-cases2.png)

### Tooltip

El componente tiene un tooltip que se puede deshabilitar. Se utiliza para indicar qué contenido se copiará.
