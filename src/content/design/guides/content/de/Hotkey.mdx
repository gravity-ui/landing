Eine modifizierte Beschriftung, die eine Kombination von Hotkeys enthält, die einer Schnittstellenaktion zugewiesen sind.

## Plattformen /. **platform**

![Group 2087325957.png](/static/images/design/Hotkey/Group_2087325957.png)

1. **Mac** – für Tastaturen von iOS-Geräten
2. **PC** – für andere Tastaturen

### Struktur

1. **Taste** — ein Symbol auf der Tastatur, das gedrückt werden muss, um eine schnelle Aktion auszuführen. Kann ein Buchstabe, eine Zahl, ein spezielles Tastatursymbol oder ein beschreibendes Wort für eine Aktion sein (z. B. Klicken). Stellen Sie sicher, dass die Symbole mit den Symbolen auf der Tastatur übereinstimmen.
   Textstil: Text/Body1, Farbe: Text/Light Complementary.
2. **Plus** — wird für die Zeichenkommunikation und die Aufrechterhaltung der richtigen Reihenfolge der Tastendrücke bei Bedarf verwendet.
   Textstil: Text/Body1, Farbe: Text/Light Hint.
