Eine Komponente zur Anzeige eines Avatars zusammen mit dem Benutzernamen und den Kontaktdaten, mit einfach zu modifizierendem Inhalt.

# Struktur

![Specs3.png](/static/images/design/User/Specs3.png)

1. Benutzeravatar (Avatar-Komponente)
2. Benutzername-Block (Optional)
3. Kontakt-Block (Optional)

# Größe

![7.png](/static/images/design/User/7.png)

**XL** - 50x50px Avatar + Textrahmen, der an die Inhaltsbreite angepasst werden kann

**L** - 42x42px Avatar + Textrahmen, der an die Inhaltsbreite angepasst werden kann

**M** - 32x32px Avatar (Standard) + Textrahmen, der an die Inhaltsbreite angepasst werden kann

**S** - 28x28px Avatar + Textrahmen, der an die Inhaltsbreite angepasst werden kann

**XS** - 24x24px Avatar + Textrahmen, der an die Inhaltsbreite angepasst werden kann

# Inhaltstypen

![11.png](/static/images/design/User/11.png)

**Vollständiger Block (Standard)**

Standardmäßig ist die Komponente so konfiguriert, dass sie die maximale Menge an Inhalt anzeigt, einschließlich eines Avatars, eines Namensbereichs und eines Kontaktbereichs.

**Anzeige ohne Benutzeravatar**

Optional kann der Benutzeravatar-Block deaktiviert werden.

# Funktionalität

![10.png](/static/images/design/User/10.png)

**Flexible Konfiguration der optionalen Anzeige**

Jeder Komponentenblock kann bei Bedarf deaktiviert werden.
