## Verwendung

1. Öffnen Sie den Link
2. Klicken Sie auf `In Figma öffnen`
3. Wenn Sie zu mehreren Organisationen eingeladen wurden, wählen Sie aus, wohin die Bibliothek kopiert werden soll
4. Verschieben Sie die Bibliothek aus Ihren Entwürfen in Ihr Projekt, passen Sie die gewünschten Stile an und veröffentlichen Sie sie
5. Fügen Sie die Bibliothek mit der `Team Library`-Oberfläche zu einer gewünschten Datei hinzu oder machen Sie sie im Admin-Bereich zur Standardbibliothek für alle Team-Dateien. Wenn Sie sie zum ersten Mal veröffentlichen, kann Figma eine Weile dauern, bis alle Stile und Komponenten zusammengestellt sind.
