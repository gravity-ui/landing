`UserLabel` — компонент для отображения профиля пользователя или электронной почты.

# Структура

![79.png](/static/images/design/UserLabel/79.png)

➊ Контейнер аватара (может содержать изображение, иконку или инициалы).

➋ Текстовый контейнер (почта, имя или логин пользователя).

➌ Иконка закрытия.

# Размер

![80.png](/static/images/design/UserLabel/80.png)

**XL** — аватар 50x50 px + текстовый фрейм, адаптируемый к ширине контента.

**L** — аватар 42x42 px  + текстовый фрейм, адаптируемый к ширине контента.

**M** — аватар 32x32 px (Default) + текстовый фрейм, адаптируемый к ширине контента.

**S** — аватар 28x28 px + текстовый фрейм, адаптируемый к ширине контента.

**XS** — аватар 24x24 px + текстовый фрейм, адаптируемый к ширине контента.

# Применимость

![86.png](/static/images/design/UserLabel/86.png)

**Тема**

Компонент поддерживает стилизацию в двух темах. В теме `Сlear` у компонента отключаются контурная обводка, фон компонента и иконки.

![87.png](/static/images/design/UserLabel/87.png)

**Тип иконки**

По умолчанию компонент поддерживает несколько типов иконок: аватары пользователей, почтовую иконку или буквенное обозначение. Опционально иконка может быть изменена на кастомную.

![88.png](/static/images/design/UserLabel/88.png)

**Действия по клику**

Компонент поддерживает запуск кастомных событий по клику на аватар с текстовой областью компонента и вызываемых по клику на иконку крестика.
