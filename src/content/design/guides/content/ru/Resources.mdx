## Порядок использования

1. Перейдите по ссылке.
2. Нажмите `Open in Figma`.
3. Если вы приглашены в несколько организаций, выберите ту, в которую вы хотите скопировать библиотеку.
4. Переместите библиотеку из черновиков в проект, настройте необходимые стили и опубликуйте ее.
5. Добавьте библиотеку в нужный файл с помощью интерфейса `Team Library` или сделайте ее библиотекой по умолчанию для всех общих файлов команды в панели администратора. При первой публикации библиотеки Figma может потребоваться некоторое время для сборки всех стилей и компонентов.
