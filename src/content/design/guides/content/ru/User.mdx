Компонент для отображения аватара совместно с именем пользователя и контактной информацией с гибкой настройкой контента.

# Структура

![Specs3.png](/static/images/design/User/Specs3.png)

1. Аватар пользователя (компонент `Avatar`).
2. Опциональный блок имени пользователя.
3. Опциональный блок контактов.

# Размер

![7.png](/static/images/design/User/7.png)

**XL** — аватар 50x50 px + текстовый фрейм, адаптируемый к ширине контента.

**L** — аватар 42x42 px  + текстовый фрейм, адаптируемый к ширине контента.

**M** — аватар 32x32 px (Default) + текстовый фрейм, адаптируемый к ширине контента.

**S** — аватар 28x28 px + текстовый фрейм, адаптируемый к ширине контента.

**XS** — аватар 24x24 px + текстовый фрейм, адаптируемый к ширине контента.

# Виды контента

![11.png](/static/images/design/User/11.png)

**Полный блок (по умолчанию)**

По умолчанию компонент отображается с максимальным наполнением, состоящим из пользовательского аватара, раздела имени и раздела контактов.

**Отображение без пользовательского аватара**

Блок пользовательского аватара опционально может быть отключен.

# Функциональность

![10.png](/static/images/design/User/10.png)

**Гибкая настройка опционального отображения**

Абсолютно любой блок компонента может быть опционально отключен.
