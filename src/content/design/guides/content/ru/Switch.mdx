## Описание

Компонент представляет собой интерактивный элемент управления, который позволяет переключаться между двумя состояниями: включено (`on`) и выключено (`off`). Этот компонент используется для включения или отключения определенных функций, опций или параметров.

![Description](/static/images/design/Switch/description.png)

## Размеры

Компонент `Switch` может иметь два основных размера:

![Sizes](/static/images/design/Switch/Sizes.png)

- **M (Medium)** — рекомендуется для стандартных случаев использования. Он обеспечивает удобное и наглядное взаимодействие для большинства пользователей.
- **L (Large)** — используется, когда необходимо выделить этот элемент или когда он будет применяться в контексте, где большой размер удобнее для восприятия.

## Состояния

Компонент `Switch` имеет четыре основных состояния:

![States](/static/images/design/Switch/States.png)

1. **Выключено (`Off`)** — неактивное состояние, когда опция или функция выключена.
2. **Включено (`On`)** — активное состояние, когда опция или функция включена.
3. **Наведение (Hover)** — когда пользователь наводит указатель мыши на элемент, он должен реагировать визуально для указания возможности взаимодействия.
4. **Заблокировано (`Disabled`)** — состояние, когда элемент неактивен и не доступен для взаимодействия.

## Свойства

У компонента есть возможность включать/выключать лейбл. По умолчанию лейбл выключен, но его можно включить с помощью свойства `Label`.

Обратите внимание, что при изменении размера элемента размер лейбла также меняется:

![Labels](/static/images/design/Switch/Labels.png)

- для размера `M` шрифт лейбла равен 13 px;
- для размера `L` шрифт лейбла равен 15 px.

## Рекомендации по использованию

Для достижения единообразия и согласованности в интерфейсе используйте переключатель, когда его включение/отключение ведёт к мгновенному изменению каких-либо опций либо состояния на странице. Мы не рекомендуем использовать переключатель в сочетании с кнопкой подтверждения действия. В таком случае лучше использовать чекбокс.
