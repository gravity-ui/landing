Модифицированный лейбл с комбинацией горячих клавиш, которые присвоены действию в интерфейсе.

## Платформы / **platform**

![Group 2087325957.png](/static/images/design/Hotkey/Group_2087325957.png)

1. **Mac** — для клавиатуры устройств на iOS.
2. **PC** — для других клавиатур.

### Структура

1. **Клавиша** — обозначение символа, который нужно нажать на клавиатуре для совершения быстрого действия. Может обозначаться буквой, цифрой, специальным символом клавиатуры или словом с описанием действия (например, `Click`). Следите за тем, чтобы символы в точности повторяли символы с клавиатуры.
   Стиль текста Text/Body1, цвет Text/Light Complementary.
2. **Плюс** – используется для связи между символами, поддерживает правильный порядок нажатия клавиш, если он важен.
   Стиль текста Text/Body1, цвет Text/Light Hint.

## Примеры использования

- **В [ActionTooltip](/design/guides/action-tooltip)** — для обозначения горячих клавиш, которые присвоены действию конкретной кнопки.

![example](/static/images/design/Hotkey/example1.png)

- **В отдельной панели горячих клавиш**, где собраны все их варианты, доступные в сервисе.

![example](/static/images/design/Hotkey/example2.png)

- В [List](/design/guides/list-item) кнопки действий, если какому-то из них присвоены горячие клавиши.
