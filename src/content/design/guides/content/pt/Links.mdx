Um elemento clicável com funcionalidade de navegação. Ao ser clicado, leva o usuário para outra página de um serviço externo ou interno. Menos frequentemente, navega o usuário para uma seção diferente da página.

Possui três tipos: normal (acento), primário (preto) e secundário (cinza).

**Normal**

É o padrão de usuário mais comum e conhecido. Usado para distinguir visualmente elementos dentro de textos, tabelas e como parte da navegação. Também é usado em páginas de erro e estados nulos.

**Primário**

Usado quando é óbvio para o usuário que um elemento é interativo, mas links com cores de acento sobrecarregarão visualmente a interface e perturbarão o equilíbrio dos elementos da página.

**Secundário**

Semelhante aos links primários, este tipo é usado quando é óbvio para o usuário que o elemento é interativo e o objetivo não é distrair o usuário dos elementos de ênfase da página. Geralmente usado em breadcrumbs ou ao exibir atributos secundários.

## **Estados**

Links sempre têm dois estados: padrão e hover.

![Frame 507123500.jpg](/static/images/design/Links/Frame_507123500.png)

Você pode habilitar a marcação para links visitados, se necessário.

## **Pode ser usado como:**

- elementos autônomos
- parte do texto

Pode incluir um ícone para indicar um link externo

![Frame 507123501.jpg](/static/images/design/Links/Frame_507123501.png)

## Tamanho

A fonte usada para links deve ser proporcional ao texto circundante e alinhar-se com a hierarquia geral da página.

## Recomendações

O texto do link deve ser autoexplicativo, mesmo quando usado sem o restante do texto, e informar ao usuário para onde o link leva (nome do serviço, título ou descrição da seção).
