## Como usar

1. Abra o link
2. Clique em `Abrir no Figma`
3. Se você foi convidado para várias organizações, selecione para onde deseja que a biblioteca seja copiada
4. Mova a biblioteca de seus rascunhos para o seu projeto, ajuste os estilos que desejar e publique-a
5. Adicione a biblioteca a um arquivo desejado usando a interface `Team Library` ou torne-a uma biblioteca padrão para todos os Arquivos da Equipe no Painel de Administração. Na primeira vez que você a publicar, o Figma pode demorar um pouco para compilar todos os estilos e componentes.
