A tipografia é baseada no uso de duas fontes e um conjunto de estilos que incluem fonte, peso, tamanho da fonte e entrelinhamento.

Por padrão, são usadas duas fontes: Inter para tipografia padrão e títulos, e Menlo como fonte monoespaçada. A fonte Inter está disponível no [Google Fonts](https://fonts.google.com/specimen/Inter?query=inter), e a Menlo é pré-instalada em alguns sistemas operacionais. Você pode alterar essas fontes de acordo com as diretrizes da sua marca.

# Parâmetros da Fonte

### Tamanho e Entrelinhamento

O entrelinhamento determina o espaço entre as linhas de base, e o tamanho da fonte estabelece o tamanho da fonte. Esses dois parâmetros determinam a legibilidade do texto.

### Peso

O peso é especificado para estilos de texto (o padrão para os grupos Body e Caption é 400) e estilos de exibição (o padrão para Subheader, Header e Display é 600). Por padrão, os estilos de exibição têm mais peso. No entanto, pode haver exceções.

# Grupos de Estilo

### **Body**

Usado para todos os tipos de texto. A fonte Inter é definida como padrão (400).

![Body](/static/images/design/Typography/Body.png)

O Body 1 Short tem entrelinhamento reduzido e é usado em blocos de conteúdo estreitos (até 300–400px de largura).

### **Caption**

As menores fontes de todas as fontes disponíveis. Usado como estilos para legendas, notas de rodapé e qualquer informação adicional. A fonte Inter é definida como padrão (400).

![Caption](/static/images/design/Typography/Caption.png)

### **Header**

Mais comumente usado como títulos em interfaces onde a densidade da informação é crucial. A fonte Inter é definida como padrão (600).

![Header](/static/images/design/Typography/Header.png)

### **Subheader**

Pode ser usado como subtítulos ou como acentos quando combinado com estilos de corpo apropriados. A fonte Inter é definida como padrão (600).

![Subheader](/static/images/design/Typography/Subheader.png)

### **Display**

Adequado para situações em que a exibição é mais importante do que a densidade da informação, como em design de comunicação. A fonte Inter é definida como padrão (600).

![Display](/static/images/design/Typography/Display.png)

### **Code**

Usado para exibir código em blocos separados. A fonte Menlo é definida como padrão.

![Code](/static/images/design/Typography/Code.png)

### **Code Inline**

Usado para exibir código que está incorporado em texto simples. O espaçamento de linha reduzido é usado para compensação óptica e alinhamento com texto simples. Ao alterar as fontes, certifique-se de que o código inline seja exibido corretamente. A fonte Menlo é definida como padrão.

![Code Inline](/static/images/design/Typography/CodeInline.png)
