O módulo é a unidade fundamental de medida no sistema de design. O módulo espacial é a base para a construção da grade e para a determinação dos tamanhos de componentes e espaçamentos. O módulo permite padronizar os princípios de especificação de espaçamentos e tamanhos dentro do sistema de design, além de alcançar consistência visual entre layouts no Figma e o código de produção final.

# Módulo

Para moldar e medir o espaço em nosso sistema de design, utilizamos o módulo de 4x4 px. Todos os espaçamentos de contêineres e componentes são calculados usando este tamanho de espaço. Em casos raros, utilizamos um micro-módulo de 2x2 px.

# Sistema de espaçamento

Utilizamos um conjunto limitado de espaçamentos tokenizados que ajudam a criar layouts limpos e consistentes. Ao criar interfaces e componentes em um sistema de design, utilize os tokens do Figma pré-configurados.

**Espaçamento interno e externo**

Espaçamento interno e externo, simétrico ou assimétrico, a partir das quatro bordas do contêiner até o conteúdo.

**Pilha horizontal e vertical**

Em pilhas horizontais e verticais, o espaçamento entre os elementos dentro do contêiner define suas relações verticais e horizontais.
