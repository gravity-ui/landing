Aqui está a tradução do arquivo fonte do guia de design para português, mantendo a naturalidade e os elementos técnicos:

Gravity-UI é uma coleção de bibliotecas e componentes para construir interfaces de usuário funcionais. O sistema de design suporta múltiplos temas: variações **clara**, **escura** e de **alto contraste** de cada um. O estilo visual dos componentes em cada tema pode ser facilmente personalizado.

**Cor**

O sistema de design consiste em quatro temas baseados em um modelo de cores de duas camadas. O motor de temas suporta personalização versátil de cores e identidades de marca com diferentes conjuntos de requisitos.

**Tipografia**

O sistema de tipografia consiste em estilos de composição, fontes de exibição e código (fonte monoespaçada). Você pode usar as fontes padrão ou criar as suas próprias.

**Controlar o raio do canto**

O raio do canto é uma parte pequena, mas importante, do estilo visual. Para facilitar o trabalho com ele, desenvolvemos um conjunto de variáveis de raio. Essas variáveis são usadas por diferentes controles. Para ajustar o raio dos controles, basta alterar os valores das variáveis.

**Criando e usando um tema**

Independentemente da marca e da linguagem visual, tudo se resume a um conjunto de constantes que você deve ser capaz de editar e usar facilmente tanto no seu código quanto na biblioteca do Figma.
