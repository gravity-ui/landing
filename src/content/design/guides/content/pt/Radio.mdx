Usado quando você precisa escolher entre várias opções mutuamente exclusivas.

### Quando usar

Semelhante ao checkbox, o Radio não aciona uma ação imediatamente. Para isso, você deve clicar no botão de confirmação (Salvar / Aplicar / Enviar).

O texto explicativo no rótulo deve responder à pergunta: "Como?" ou "Qual?".

**Quando não usar**

O Radio não pode ser usado para alternar entre conteúdo de página ou menu. Essa funcionalidade é suportada pelos componentes [radio-button](https://gravity-ui.com/design/guides/radio-button) e tab.

### Estrutura

![Estrutura](/static/images/design/Radio/radio-content.png)

1 — Elemento Radio, 2 — rótulo explicativo

O Radio é sempre usado com um rótulo que explica seu significado.

A primeira opção é selecionada por padrão.

Você também pode deixar os controles de alternância desmarcados (quando todas as opções são removidas).

Uma vez que um item dentro de um grupo foi selecionado, o usuário não pode desmarcá-lo para retornar o grupo ao seu estado original (sem atualizar a página).

Ao usar um componente, recomendamos que você sempre selecione a primeira opção da lista.

### Tamanhos

Vem em dois tamanhos: M e L. O tamanho do componente é determinado pelo tamanho dos controles da página. Por exemplo, se a página usa o tamanho L para inputs, botões e outros elementos, então o radio deve ter o mesmo tamanho.

### Estados

![Estados](/static/images/design/Radio/radio-state.png)

Padrão – quando o usuário não interage com o elemento

Hover – ao passar o mouse sobre um elemento e um rótulo de texto

Marcado – estado selecionado

Desabilitado – estado desabilitado, quando a opção selecionada não está disponível para interação
