**Tooltip** — [Action Tooltip](/design/guides/action-tooltip)의 기반이 되는 기본 컴포넌트입니다. 버튼 위에 마우스를 올렸을 때 표시되며, 버튼 클릭 시 수행될 작업에 대한 설명이나 추가 정보를 포함합니다.

툴팁은 힌트 내부에 사용자 정의 콘텐츠를 포함해야 할 경우에만 사용됩니다. 다른 경우에는 [Action Tooltip](/design/guides/action-tooltip)을 사용하는 것이 좋습니다.

### 크기 및 스타일

툴팁의 크기는 콘텐츠에 맞춰 조정됩니다. 툴팁의 색상과 스타일은 변경할 수 없습니다.

![Sizes](/static/images/design/Tooltip/Sizes.png)

### 내부 콘텐츠

툴팁은 모든 유형의 콘텐츠를 포함할 수 있습니다. 툴팁 콘텐츠를 디자인할 때는 일반적인 가이드라인과 들여쓰기를 따르는 것이 좋습니다.

### 타이밍

목적에 따라 표시 및 사라지는 타이밍을 변경할 수 있습니다. 툴팁은 기본적으로 250ms의 지연 후 나타나며, 사용자가 버튼에서 마우스를 떼면 사라집니다.

대부분의 사용자가 툴팁을 읽지 않고 버튼을 클릭할 것이라고 생각된다면, 너무 많은 프롬프트로 사용자를 귀찮게 하는 대신 지연 시간을 사용하는 것이 좋습니다. 툴팁 없이 버튼을 클릭하는 사람이 적을 것이라고 생각된다면, 사용자가 기능 설명을 기다리지 않도록 지연 시간을 제거하세요.

### 위치

툴팁의 위치는 페이지 콘텐츠와의 겹침을 최소화하는 것을 목표로 선택됩니다. 툴팁의 최적 배치는 페이지 콘텐츠와의 겹침이 가장 적은 곳입니다.
