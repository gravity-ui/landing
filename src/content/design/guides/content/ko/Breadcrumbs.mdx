**브레드크럼**은 서비스 구조에서 루트 페이지부터 현재 페이지까지의 경로를 나타내는 링크 체인으로, 사용자가 중간의 어떤 페이지로든 이동할 수 있도록 합니다. 이를 통해 사용자는 자신이 서비스의 어느 부분에 있는지 파악하고 한두 단계 뒤로 빠르게 이동할 수 있습니다.

![Example](/static/images/design/Breadcrumbs/Example.png)

## 사용 시기

- 중첩된 페이지에 브레드크럼 컴포넌트를 사용하세요.
- 홈 페이지에서는 브레드크럼을 사용할 필요가 없습니다.
- 브레드크럼은 일반적으로 페이지 맨 위 액션 바에 배치됩니다.

## 구조

![Structure](/static/images/design/Breadcrumbs/Structure.png)

1 - **현재 페이지.**

2 - **이전 페이지.** 컨테이너 너비가 허용하는 경우 가장 먼저 표시하는 것이 좋습니다.

3 - **서비스의 루트 페이지.**

4 - **구분자.** 기본적으로 "/"가 사용되지만, ">"도 허용됩니다.

5 - **오버플로우 아이콘.** 페이지 제목이 컨테이너 너비에 맞지 않을 때 나타납니다. 사용 가능한 공간보다 더 많은 공간을 차지하는 페이지는 줄임표를 클릭하면 나타나는 드롭다운 메뉴를 통해 접근할 수 있습니다.

## 크기

브레드크럼의 기본 글꼴 크기는 body-1입니다. 아이콘 크기는 16x16px입니다.

![Sizes](/static/images/design/Breadcrumbs/Sizes.png)

## 상태

![States](/static/images/design/Breadcrumbs/States.png)

**기본** - 기본 상태입니다.

**호버** - 페이지 제목 위에 마우스를 올렸을 때입니다.

**비활성화** - 클릭할 수 없습니다. 현재 페이지 또는 연결되지 않은 페이지에 사용할 수 있습니다.

**로딩** - `renderItemContent` 매개변수를 통해 함수를 전달하여 컴포넌트가 요소의 콘텐츠를 표시할 수 있습니다. 예를 들어, 크럼 요소의 데이터가 아직 로드되지 않은 경우 Skeleton을 표시할 수 있습니다.

## 수정

![Modifications](/static/images/design/Breadcrumbs/Modifications1.png)

브레드크럼에는 다음과 같은 추가 요소가 포함될 수 있습니다.

1. **액션 아이콘 -** 즐겨찾기 추가, 페이지 제목 복사 또는 컨텍스트 메뉴와 같은 기능입니다.
2. **[레이블](/design/guides/label)** — 페이지와 관련된 상태 또는 다른 속성입니다.
3. **선택** — 요소를 클릭하면 다양한 옵션 중에서 선택할 수 있습니다.
4. **추가 텍스트**—ID, 수량 카운터 및 기타 페이지 속성

홈 페이지로 돌아가기 위해 "메인"이라는 단어 대신 집 아이콘을 사용할 수 있습니다.

![Modifications](/static/images/design/Breadcrumbs/Modifications1.png)

## 사용자 정의

색상, 구분자 및 호버 스타일을 변경할 수 있습니다. 현재 페이지에 더 밝은 강조 표시를 유지하고 요소 간에 충분한 대비를 유지하는 것을 잊지 마세요.
