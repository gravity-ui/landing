## 설명

Tabs 컴포넌트를 사용하면 사용자가 전환하여 볼 수 있는 여러 섹션으로 콘텐츠를 나누어 구성할 수 있습니다. 탭을 사용하여 관련 섹션을 그룹화하고 각 섹션에 빠르게 액세스할 수 있습니다.

![1.png](/static/images/design/Tabs/tabs-0001.png)

## 크기

Tabs는 세 가지 크기를 가질 수 있습니다.

![2.png](/static/images/design/Tabs/tabs-0002.png)

- **M (Medium):** 페이지 내의 일반적인 섹션에 이 크기를 사용합니다.
- **L (Large):** 탭이 페이지의 하위 제목 역할도 하는 경우 L 크기를 사용합니다. 이 경우 탭 콘텐츠는 페이지의 별도 섹션이며 탭은 이 섹션의 하위 제목입니다.
- **XL (Extra Large):** 탭이 페이지 제목 역할도 하는 경우 이 크기가 적합합니다. 이 경우 사용자가 탭을 전환하면 전체 페이지의 콘텐츠가 변경됩니다.

## 상태

Tabs는 다음과 같은 상태를 가질 수 있습니다.

![3.png](/static/images/design/Tabs/tabs-0003.png)

- **비활성 (Inactive):** 탭이 선택되지 않은 상태입니다. 이 상태에서는 활성 탭을 강조하기 위해 탭 색상이 더 흐릿하게 표시됩니다.
- **활성 (Active):** 현재 활성 콘텐츠 섹션을 표시하는 선택된 탭에 할당되는 상태입니다. 활성 탭은 더 채도가 높은 색상을 가지며 강조를 위해 밑줄이 그어집니다.
- **호버 (Hover):** 사용자가 탭 위로 마우스를 가져가면 포인터 스타일이 변경되어 상호 작용 가능성을 나타냅니다.
- **비활성화 (Disabled):** 탭을 선택할 수 없거나 일시적으로 또는 영구적으로 사용할 수 없는 경우 비활성화될 수 있습니다. 이러한 탭은 회색으로 표시되며 상호 작용할 수 없습니다.

## 속성

각 탭에 대해 다음 요소를 활성화/비활성화할 수 있습니다.

![4.png](/static/images/design/Tabs/tabs-0004.png)

1. **아이콘 (Icon):** 탭 내부 콘텐츠의 유형을 시각화하기 위해 디자이너의 재량에 따라 사용됩니다.
2. **레이블 (Label):** 텍스트 형식의 섹션 이름입니다.
3. **카운터 (Counter):** 탭 내부의 항목 수를 나타내는 데 사용할 수 있습니다. 또한 탭에 새로 생긴 항목 수를 지정하는 데 사용할 수도 있습니다.
4. **레이블 (Label):** 탭이 담당하는 섹션의 상태를 나타내는 데 사용할 수 있습니다.

## 사용 권장 사항

Tabs를 사용하여 대량의 콘텐츠를 구성하거나 정보를 논리적인 섹션으로 나누십시오. 탭 제목에 명확하고 읽기 쉬운 구조를 사용하십시오.

프로젝트의 전체 디자인과 일치하는 탭 스타일과 색상을 선택하십시오. 사용자가 현재 섹션을 쉽게 식별할 수 있도록 항상 활성 탭을 강조 표시하십시오.

가능한 경우 M 및 L 탭 섹션이 단일 화면 높이에 맞도록 하십시오. 이렇게 하면 콘텐츠와 탭 이름을 쉽게 연결하고 탭 콘텐츠가 끝나고 다음 블록이 시작되는 위치를 볼 수 있습니다.
