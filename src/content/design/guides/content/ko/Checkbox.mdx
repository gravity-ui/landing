체크박스는 사용자에게 제공된 여러 옵션 중에서 하나 이상을 선택할 수 있도록 하는 사용자 인터페이스 요소입니다.

## 사용 사례

![Cases](/static/images/design/Checkbox/Checkbox1.png)

활성화 시 체크박스는 즉시 동작을 트리거하지 않습니다. 사용자는 이를 적용하기 위해 저장, 적용, 제출과 같은 확인 버튼을 클릭해야 합니다. 대부분의 경우 체크박스는 설명 레이블과 함께 사용됩니다. 그러나 테이블이나 각 체크박스 옆에 "활성화" 레이블을 포함하는 것이 불필요한 경우와 같이 레이블 없이 사용될 수도 있습니다. 사용자가 논리 퍼즐을 푸는 것을 방지하기 위해 "비활성화" 대신 "활성화"와 같이 긍정적인 표현으로 텍스트를 시작하세요.

## 구조

체크박스는 다음과 같이 구성됩니다.

![Anatomy](/static/images/design/Checkbox/Checkbox2.png)

1.  **컨테이너**: 체크박스를 둘러싸는 시각적인 사각형 영역입니다. 컨테이너는 경계를 나타내고 체크박스를 활성화합니다.
2.  **체크 표시**: 사용자가 체크박스를 활성화할 때 컨테이너 내에 나타나는 기호입니다. 이 기호는 사용자가 체크박스를 선택했음을 시각적으로 확인합니다.
3.  **텍스트 레이블 (선택 사항)**: 체크박스 오른쪽에 배치되는 텍스트입니다. 텍스트 레이블은 체크박스의 의미나 목적을 설명합니다.

## 크기

체크박스 컴포넌트는 두 가지 주요 크기를 가집니다.

![Sizes](/static/images/design/Checkbox/Checkbox3.png)

**M**은 기본 크기로, 대부분의 경우에 적합한 표준 치수를 가집니다. 이 크기에서 컨테이너는 14x14px이고 텍스트 레이블은 13px 글꼴을 사용합니다.

**L**은 체크박스가 다른 인터페이스 요소들 사이에서 눈에 띄어야 하거나 인터페이스에 더 큰 요소를 사용하여 사용 편의성을 높여야 할 때 사용됩니다. 이 크기에서 컨테이너는 17x17px이고 텍스트 레이블은 15px 글꼴을 사용합니다.

## 상태

![States](/static/images/design/Checkbox/Checkbox4.png)

**기본**: 사용자가 클릭하여 활성화하거나 비활성화할 수 있는 주요 체크박스 상태입니다.

**호버**: 사용자가 체크박스 영역(체크박스 + 레이블) 위로 마우스를 가져갔을 때의 상태입니다.

**비활성화**: 어떤 이유로든 사용자가 체크박스와 상호 작용할 수 없는 상태입니다.

**불확정**: 체크박스 그룹의 헤더 상태로, 여러 항목 중 하나가 선택되었을 때 (일반적으로 여러 행 선택이 가능한 테이블에서 발생합니다).

## 권장 사항

### 사용

-   **옵션 선택**: 목록에서 하나 이상의 옵션을 선택하도록 하려는 경우.
-   **기능 활성화/비활성화**: 인터페이스에서 특정 기능을 활성화하거나 비활성화하도록 하려는 경우. 예를 들어, 사용자는 알림 또는 자동 재생과 같은 기능을 활성화하거나 비활성화할 수 있습니다. 이러한 경우 체크박스는 명확하고 직관적인 상호 작용을 제공합니다.
-   **테이블 또는 목록에서 다중 선택**: 삭제 또는 내보내기와 같은 일괄 작업을 위해 여러 항목을 선택하도록 하려는 경우. 이는 데이터 작업을 단순화하고 일괄 작업을 용이하게 합니다.
-   **콘텐츠 또는 결과 필터링**: 콘텐츠 또는 검색 결과를 필터링하기 위한 매개변수를 선택하도록 하려는 경우.

### 피해야 할 경우

-   **단일 옵션 선택**: 상호 배타적인 세트에서 하나의 옵션을 선택해야 하는 경우 라디오 버튼을 사용하세요.
-   **두 가지 상태의 활성화/비활성화 (즉시 적용)**: 옵션이 두 가지 상태(활성화 및 비활성화)만 가지고 있고 "적용" 버튼 없이 변경 사항이 즉시 적용되는 경우 스위치를 사용하세요. 스위치는 이러한 옵션을 더 명확하고 간결하게 표시하며, 전환 후 변경 사항이 즉시 적용됩니다.
-   **세그먼트 선택**: 사용자가 다른 범주에서 동시에 선택할 수 있는 여러 상호 배타적인 옵션을 제공하는 경우 세그먼트 컨트롤을 사용하세요. 이를 통해 사용자는 다른 세그먼트 또는 범주에서 옵션을 선택할 수 있습니다.
-   **다수의 옵션에서 선택**: 단일 옵션을 선택하기 위해 긴 옵션 목록을 제공하는 경우 드롭다운 목록(select)을 사용하세요. 이는 화면 공간을 절약하고 많은 옵션 중에서 선택을 용이하게 합니다.

### 텍스트 작성

-   텍스트 설명은 간결하게 작성하세요. 긴 구문이나 복잡한 문장은 피하세요.
-   선택과 관련된 동작을 명확하게 나타내는 능동 동사를 사용하세요. 예를 들어, "뉴스레터 구독" 대신 "뉴스레터 구독하기"와 같이 사용하세요.
-   체크박스를 선택했을 때 어떤 일이 발생하는지 사용자가 완전히 이해하도록 하세요. 예를 들어, "알림" 대신 "새 글 알림 받기"와 같이 사용하세요.
