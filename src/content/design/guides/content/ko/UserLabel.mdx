# 구조

![79.png](/static/images/design/UserLabel/79.png)

➊ 아바타 컨테이너 (이미지, 아이콘 또는 이니셜)

➋ 텍스트 컨테이너 (이메일, 사용자 이름 또는 로그인)

➌ 닫기 아이콘

# 크기

![80.png](/static/images/design/UserLabel/80.png)

**XL** - 50x50px 아바타 + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

**L** - 42x42px 아바타 + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

**M** - 32x32px 아바타 (기본값) + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

**S** - 28x28px 아바타 + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

**XS** - 24x24px 아바타 + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

# 범위

![86.png](/static/images/design/UserLabel/86.png)

**테마**

컴포넌트는 두 가지 테마를 제공합니다. 'Clear' 테마에서는 컴포넌트의 외곽선, 배경 및 아이콘이 비활성화됩니다.

![87.png](/static/images/design/UserLabel/87.png)

**아이콘 유형**

컴포넌트는 기본적으로 여러 유형의 아이콘을 지원합니다. 사용자 아바타, 이메일 아이콘 또는 문자를 기반으로 한 식별자입니다. 선택적으로 아이콘을 사용자 정의 아이콘으로 교체할 수 있습니다.

![88.png](/static/images/design/UserLabel/88.png)

**클릭 시 동작**

컴포넌트에서는 컴포넌트 텍스트 영역의 아바타를 클릭하여 사용자 정의 이벤트를 시작할 수 있으며, 'X' 아이콘을 클릭하여 트리거되는 이벤트도 있습니다.
