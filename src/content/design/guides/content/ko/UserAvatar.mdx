사용자 아바타는 특정 사용자 또는 계정을 빠르게 식별하고 인증된 사용자를 표시하는 데 사용됩니다.

# 크기

![1.png](/static/images/design/UserAvatar/1.png)

**XL** - 50 x 50 픽셀

**L** - 42 x 42 픽셀

**M** - 32 x 32 픽셀 (기본값)

**S** - 28 x 28 픽셀

**XS** - 24 x 24 픽셀

<aside>
💡 이러한 표준 아바타 크기는 주변 텍스트를 읽기 쉽게 유지합니다.
</aside>

# 콘텐츠 유형

![2.png](/static/images/design/UserAvatar/2.png)

**이미지 없음 (기본값):** 이미지 소스가 없거나 로딩 오류가 발생하면 기본 플레이스홀더가 컨테이너를 채웁니다.

**업로드된 이미지:** 사용자가 이미지를 업로드하면 해당 이미지가 전체 아바타 영역의 플레이스홀더로 사용됩니다. 이미지는 기본적으로 전체 컨테이너 너비(100%)를 채웁니다.

# 기능

**클릭 또는 호버 시 동작:** 선택적으로 사용자 아바타에 클릭 또는 호버 시 표시되는 추가 정보가 포함될 수 있습니다.

![2.png](/static/images/design/UserAvatar/2%201.png)

**사용자 지정 크기 값:** 선택적으로 아바타 크기를 사용자 지정 값으로 설정할 수 있습니다. 가능한 경우 기본 크기를 사용하세요.
