이 컴포넌트를 사용하면 테이블과 같이 페이지별로 표시되는 콘텐츠를 탐색할 수 있습니다.

![IMG - 8.png](/static/images/design/Pagination/paginator-0001.png)

## 사용 시기

✅ **사용 시기**

- 페이지로 나누어 표시되는 모든 경우에 사용합니다.
- 사용자가 여러 페이지를 건너뛰거나 원하는 페이지로 바로 이동할 수 있도록 합니다.
- 사용자가 표시할 행 수를 사용자 지정할 수 있도록 합니다.
- 페이지 내 섹션의 크기를 고정하여 상단의 자동 콘텐츠 로딩이 하단 콘텐츠를 이동시키지 않도록 합니다.

![3.png](/static/images/design/Pagination/paginator-0002.png)

🚫 **사용하지 않을 때**

검색 결과 페이지 간의 원활한 전환이 필요한 경우, 페이지 속도가 느려지지 않는 한 자동 로딩 또는 버튼 로딩을 사용하세요.

## 구조

![IMG - 18.png](/static/images/design/Pagination/paginator-0005.png)

1. **현재 페이지.** 표시되는 페이지의 일련 번호를 보여줍니다.
2. **이전 페이지.** 이전 페이지를 엽니다.
3. **다음 페이지.** 다음 페이지를 엽니다.
4. **콘텐츠의 마지막 페이지.** 총 콘텐츠를 알 수 있고 즉시 페이지로 나눌 수 있을 때 표시됩니다. 클릭하면 마지막 페이지를 엽니다.
5. **오버플로우 아이콘.** 페이지 수가 네 개를 초과할 때 표시됩니다.
6. **앞/뒤 전환.** 페이지를 하나씩 전환합니다.
7. **첫 페이지로 이동.** 첫 번째 콘텐츠 페이지를 엽니다.
8. **특정 페이지로 이동.** 사용자가 페이지 번호를 입력하고 Enter 키를 누르면 지정된 페이지를 엽니다.
9. **페이지당 항목 수.** 한 페이지에 표시되는 요소 수를 전환합니다.

구조의 모든 요소를 함께 사용할 필요는 없습니다. 사용 가능한 구성은 "수정 사항"에서 확인할 수 있습니다.

## 상태

![IMG - 19.png](/static/images/design/Pagination/paginator-0007.png)

**첫 페이지**는 첫 페이지를 보여줍니다.

**두 번째/다음 페이지**는 두 번째 및 다음 페이지를 보여줍니다.

**마지막 페이지**는 마지막 페이지를 보여줍니다.

## 수정 사항

![IMG - 20.png](/static/images/design/Pagination/paginator-0008.png)

1. **페이지 크기 옵션 및 입력란 표시**: 사용자가 특정 페이지로 이동하거나 페이지에 표시되는 콘텐츠 양을 제어할 수 없는 경우 이러한 필드를 비활성화할 수 있습니다.
2. **총계 알 수 없음**은 현재 페이지만 표시합니다. 총 요소 수를 알 수 없을 때 사용합니다.
3. **페이지 숨기기**는 일련 번호 없이 페이지 전환을 보장합니다. 요소 목록이 동적이고 페이지 수와 일련 번호가 계속 변경되어 사용자를 혼란스럽게 할 때 사용합니다.
4. **컴팩트**는 텍스트가 없는 컴팩트 형식입니다. 인터페이스 공간이 제한적일 때 사용합니다(예: 대화 상자).

모바일 장치 인터페이스의 경우 모바일 버전 페이저를 사용할 수 있습니다.
