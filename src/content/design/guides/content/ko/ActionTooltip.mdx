버튼에 마우스를 올렸을 때 나타나는 툴팁입니다. 일반적으로 버튼을 클릭했을 때 수행될 작업에 대한 설명이나 추가 정보를 포함합니다. 아이콘만 있는 버튼에 자주 사용되어 사용자에게 해당 아이콘과 관련된 작업에 대한 명확한 이해를 제공합니다.

## 구조

![2.png](/static/images/design/ActionTooltip/Structure.png)

1. **제목**은 툴팁에서 가장 중요하고 필수적인 요소이며, 버튼의 주요 기능을 설명합니다. 제목 텍스트는 한 줄에 맞도록 작성하는 것을 권장합니다.
2. **설명**은 제목에 대한 추가 정보를 제공하는 선택적 툴팁 요소입니다.
3. **단축키 / 바로가기**는 인터페이스에서 설명된 작업을 시작하는 키 조합으로, 선택적 툴팁 요소입니다. 사용자가 서비스와의 상호 작용을 빠르게 할 수 있도록 빠른 액션 방법을 알려주는 데 사용됩니다.

## 옵션

![Group 2087325956.png](/static/images/design/ActionTooltip/Variants.png)

1. **제목만**은 가장 흔하게 사용되는 옵션으로, 툴팁이 작업 자체만을 간략하게 설명합니다. 텍스트는 5단어 이하로 작성하는 것을 권장합니다.
2. **제목과 단축키**는 작업에 할당된 키 조합이 있는 경우 사용자 안내를 위해 표시하는 것이 좋습니다.
3. **제목과 설명**은 작업이 복잡하여 제목을 간결하고 명확하게 작성하기 어려운 경우, 추가 정보를 포함하는 설명을 포함하는 것이 좋습니다.
4. **제목, 설명, 단축키**는 인터페이스에서 사용자가 결정을 내리는 데 모든 요소가 필요한 경우 사용되는 최대 구성입니다.

## 크기

단일 제목 또는 제목+단축키를 사용하는 경우, 툴팁 너비는 텍스트에 맞춰 조정됩니다. 설명의 경우 너비는 300px로 설정됩니다.

모든 옵션의 패딩: 좌우 12px, 상하 8px.

단축키가 있는 옵션에서는 제목과 레이블 사이의 간격이 8px로 설정됩니다.

설명이 있는 옵션(단축키 유무에 관계없이)에서는 제목과 설명 사이의 거리가 2px입니다.

![158.png](/static/images/design/ActionTooltip/Sizes.png)

## 위치

버튼과의 관계에서 툴팁의 기본 위치는 하단 중앙이며, 요소 간 거리는 4px입니다.

![13.png](/static/images/design/ActionTooltip/LocationUnder.png)

버튼이 페이지의 왼쪽 또는 오른쪽 가장자리에 위치하는 경우, 툴팁은 해당 버튼 가장자리에 맞춰 정렬됩니다.

![14.png](/static/images/design/ActionTooltip/LocationUnderLeft&right.png)

버튼이 화면 하단에 있는 경우, 툴팁은 버튼의 위치에 해당하는 상단에 표시됩니다: 왼쪽 가장자리, 중앙 또는 오른쪽 가장자리.

![15.png](/static/images/design/ActionTooltip/LocationAboveLeft&center&right.png)

## 타이밍

목적에 따라 마우스를 올렸을 때 툴팁이 나타나고 사라지는 지연 시간을 조정할 수 있습니다. 대부분의 경우 툴팁은 즉시 또는 최대 250ms의 최소 지연 시간으로 나타나며, 사용자가 버튼에서 마우스를 떼면 사라집니다.

대부분의 사용자가 힌트 없이 버튼의 기능을 이해할 수 있다고 생각되면, 툴팁은 어려움을 겪는 사용자에게만 표시하고 불필요한 요소로 사용자를 짜증나게 하지 않도록 나타나는 지연 시간을 늘릴 수 있습니다.
