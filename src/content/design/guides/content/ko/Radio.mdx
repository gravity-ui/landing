여러 개의 상호 배타적인 옵션 중에서 선택해야 할 때 사용합니다.

### 언제 사용해야 하나요?

체크박스와 유사하게, 라디오 버튼은 즉시 동작을 트리거하지 않습니다. 이를 위해서는 확인 버튼(저장 / 적용 / 전송)을 클릭해야 합니다.

레이블의 설명 텍스트는 "어떻게?" 또는 "어떤 것을?"이라는 질문에 답해야 합니다.

**언제 사용하지 않아야 하나요?**

라디오 버튼은 페이지 또는 메뉴 콘텐츠를 전환하는 데 사용할 수 없습니다. 이 기능은 [radio-button](https://gravity-ui.com/design/guides/radio-button) 및 탭 컴포넌트에서 지원합니다.

### 구조

![구조](/static/images/design/Radio/radio-content.png)

1 — 라디오 버튼 요소, 2 — 설명 레이블

라디오 버튼은 항상 그 의미를 설명하는 레이블과 함께 사용됩니다.

첫 번째 옵션이 기본적으로 선택됩니다.

토글 컨트롤을 선택되지 않은 상태로 둘 수도 있습니다 (모든 옵션이 제거된 경우).

그룹 내에서 항목이 선택되면 사용자는 해당 항목을 선택 해제하여 그룹을 원래 상태(페이지 새로고침 없이)로 되돌릴 수 없습니다.

컴포넌트를 사용할 때는 목록의 첫 번째 옵션을 항상 선택하는 것이 좋습니다.

### 크기

M과 L 두 가지 크기로 제공됩니다. 컴포넌트의 크기는 페이지 컨트롤의 크기에 따라 결정됩니다. 예를 들어, 페이지에서 입력란, 버튼 및 기타 요소에 L 크기를 사용하는 경우 라디오 버튼도 같은 크기여야 합니다.

### 상태

![상태](/static/images/design/Radio/radio-state.png)

기본 — 사용자가 요소와 상호 작용하지 않을 때

호버 — 요소와 텍스트 레이블 위에 마우스를 올렸을 때

선택됨 — 선택된 상태

비활성화 — 선택된 옵션을 상호 작용할 수 없을 때의 비활성화된 상태
