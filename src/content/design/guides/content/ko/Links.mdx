## 링크 (Link)

사용자에게 다른 페이지로 이동하는 네비게이션 기능을 제공하는 클릭 가능한 요소입니다. 클릭 시 외부 또는 내부 서비스의 다른 페이지로 이동합니다. 드물게는 페이지 내 다른 섹션으로 이동하는 데 사용되기도 합니다.

세 가지 타입이 있습니다: 일반 (accent), 기본 (primary, black), 보조 (secondary, gray).

**일반 (Normal)**

가장 흔하고 잘 알려진 사용자 패턴입니다. 텍스트, 테이블 내에서 요소를 시각적으로 구분하거나 네비게이션의 일부로 사용됩니다. 오류 페이지나 null 상태에서도 사용됩니다.

**기본 (Primary)**

사용자에게 요소가 상호작용 가능하다는 것이 명확하지만, 강조 색상의 링크가 인터페이스를 시각적으로 과부하시키고 페이지 요소의 균형을 해칠 수 있을 때 사용됩니다.

**보조 (Secondary)**

기본 링크와 유사하게, 사용자에게 요소가 상호작용 가능하다는 것이 명확하고 페이지의 강조 요소에서 사용자의 주의를 분산시키지 않는 것이 목표일 때 사용됩니다. 일반적으로 브레드크럼(breadcrumbs)이나 보조 속성을 표시할 때 사용됩니다.

## **상태 (States)**

링크는 항상 두 가지 상태를 가집니다: 기본(default) 및 호버(hovered).

![Frame 507123500.jpg](/static/images/design/Links/Frame_507123500.png)

필요한 경우 방문한 링크에 대한 태그를 활성화할 수 있습니다.

## **사용 방식:**

- 독립적인 요소
- 텍스트의 일부

외부 링크임을 나타내는 아이콘을 포함할 수 있습니다.

![Frame 507123501.jpg](/static/images/design/Links/Frame_507123501.png)

## 크기 (Size)

링크에 사용되는 폰트는 주변 텍스트와 비례해야 하며, 페이지의 전체적인 계층 구조와 일치해야 합니다.

## 권장 사항 (Recommendations)

링크 텍스트는 다른 텍스트 없이 사용될 때도 자체적으로 의미를 명확히 전달해야 하며, 사용자가 링크가 어디로 연결되는지 (서비스 이름, 제목 또는 섹션 설명) 알 수 있도록 해야 합니다.
