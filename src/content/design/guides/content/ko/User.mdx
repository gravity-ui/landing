사용자 이름 및 연락처 정보와 함께 아바타를 표시하고 콘텐츠를 쉽게 수정할 수 있는 컴포넌트입니다.

# 구조

![Specs3.png](/static/images/design/User/Specs3.png)

1. 사용자 아바타 (Avatar 컴포넌트)
2. 사용자 이름 블록 (선택 사항)
3. 연락처 블록 (선택 사항)

# 크기

![7.png](/static/images/design/User/7.png)

**XL** - 50x50px 아바타 + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

**L** - 42x42px 아바타 + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

**M** - 32x32px 아바타 (기본값) + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

**S** - 28x28px 아바타 + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

**XS** - 24x24px 아바타 + 콘텐츠 너비에 맞게 조절 가능한 텍스트 프레임

# 콘텐츠 유형

![11.png](/static/images/design/User/11.png)

**전체 블록 (기본값)**

기본적으로 컴포넌트는 아바타, 이름 섹션, 연락처 섹션을 포함하여 최대량의 콘텐츠를 표시하도록 구성됩니다.

**사용자 아바타 없이 표시**

선택적으로 사용자 아바타 블록을 비활성화할 수 있습니다.

# 기능
