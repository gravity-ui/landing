La typographie repose sur l'utilisation de deux polices et d'un ensemble de styles incluant la police, le poids, la taille de la police et l'interlignage.

Deux polices sont utilisées par défaut : Inter pour la composition standard et les titres, et Menlo comme police monospace. Inter est disponible sur [Google Fonts](https://fonts.google.com/specimen/Inter?query=inter), et Menlo est préinstallé sur certains systèmes d'exploitation. Vous pouvez modifier ces polices conformément aux directives de votre marque.

# Paramètres de police

### Taille et interlignage

L'interlignage détermine l'espace entre les lignes de base, et la taille de la police établit la taille de la police. Ces deux paramètres déterminent la lisibilité du texte.

### Poids

Le poids est spécifié pour les styles de texte (la valeur par défaut pour les groupes Body et Caption est 400) et les styles d'affichage (la valeur par défaut pour Subheader, Header et Display est 600). Par défaut, les styles d'affichage ont un poids plus important. Cependant, il peut y avoir des exceptions.

# Groupes de styles

### **Body**

Utilisé pour tous les types de texte. Inter est défini par défaut (400).

![Body](/static/images/design/Typography/Body.png)

Body 1 Short a un interlignage réduit et est utilisé dans des blocs de contenu étroits (jusqu'à 300–400px de largeur).

### **Caption**

Les plus petites polices de toutes les polices disponibles. Utilisé comme styles pour les légendes, les notes de bas de page et toute information supplémentaire. Inter est défini par défaut (400).

![Caption](/static/images/design/Typography/Caption.png)

### **Header**

Le plus souvent utilisé comme titres dans les interfaces où la densité d'information est cruciale. Inter est défini par défaut (600).

![Header](/static/images/design/Typography/Header.png)

### **Subheader**

Peut être utilisé comme sous-titres ou comme accents lorsqu'il est associé à des styles de corps appropriés. Inter est défini par défaut (600).

![Subheader](/static/images/design/Typography/Subheader.png)

### **Display**

Convient aux situations où l'affichage est plus important que la densité d'information, comme dans la conception de communication. Inter est défini par défaut (600).

![Display](/static/images/design/Typography/Display.png)

### **Code**

Utilisé pour afficher du code dans des blocs séparés. Menlo est défini par défaut.

![Code](/static/images/design/Typography/Code.png)

### **Code Inline**

Utilisé pour afficher du code intégré dans du texte brut. Un espacement réduit entre les lignes est utilisé pour une compensation optique et un alignement avec le texte brut. Lors du changement de polices, assurez-vous que le code inline s'affiche correctement. Menlo est défini par défaut.

![Code Inline](/static/images/design/Typography/CodeInline.png)
