Voici la traduction du fichier source du guide de conception en français :

Le rayon des coins des contrôles peut être remplacé. Pour ce faire, utilisez les variables suivantes :

- `--g-border-radius-xs: 3px;`
- `--g-border-radius-s: 5px;`
- `--g-border-radius-m: 6px;`
- `--g-border-radius-l: 8px;`
- `--g-border-radius-xl: 10px;`
- `--g-border-radius-2xl: 16px;`

![Corner radius](/static/images/design/CornerRadius/1.png)

Vous pouvez remplacer les valeurs dans **Variables**.

Les variables sont utilisées dans les contrôles de tailles correspondantes. Si vous modifiez la valeur d'une variable, le rayon des coins de tous les contrôles de même taille sera modifié en conséquence. Les contrôles affectés par cette variable sont listés sur la page Branding de la bibliothèque Figma. Comme les différents contrôles ont des plages de tailles différentes, les ensembles de contrôles peuvent ne pas correspondre. Les labels, par exemple, n'existent pas dans des tailles supérieures à L, donc les variables commençant par `--g-border-radius-l` ne sont pas utilisées dans ce composant et n'ont aucun effet sur son apparence.

![Corner radius](/static/images/design/CornerRadius/2.png)

Le rayon des coins de l'élément imbriqué pour le bouton de coche est déterminé par les tokens et est calculé à l'aide de la formule "valeur du token - 3" (où 3 est la distance entre les éléments internes et externes).
