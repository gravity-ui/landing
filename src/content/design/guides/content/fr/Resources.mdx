## Comment utiliser

1. Ouvrez le lien
2. Cliquez sur `Ouvrir dans Figma`
3. Si vous avez été invité dans plusieurs organisations, sélectionnez celle où vous souhaitez que la bibliothèque soit copiée
4. Déplacez la bibliothèque de vos brouillons vers votre projet, ajustez les styles souhaités et publiez-la
5. Ajoutez la bibliothèque à un fichier souhaité en utilisant l'interface `Team Library` ou faites-en une bibliothèque par défaut pour tous les fichiers de l'équipe dans le panneau d'administration. Lors de la première publication, Figma peut prendre un certain temps pour assembler tous les styles et composants.
