用户头像用于快速识别特定用户或账户，并显示授权用户。

# 尺寸

![1.png](/static/images/design/UserAvatar/1.png)

**XL** - 50 x 50 px

**L** - 42 x 42 px

**M** - 32 x 32 px (默认)

**S** - 28 x 28 px

**XS** - 24 x 24 px

<aside>
💡 这些标准的头像尺寸可确保周围文本的可读性。
</aside>

# 内容类型

![2.png](/static/images/design/UserAvatar/2.png)

**无图片 (默认):** 如果图片源丢失或发生加载错误，将使用默认占位符填充容器。

**已上传图片:** 如果用户上传了图片，该图片将用作整个头像区域的占位符。默认情况下，图片会填充整个容器宽度 (100%)。

# 功能

**点击或悬停时的操作:** 可选地，用户头像可以包含额外信息，这些信息会在点击或悬停时显示。

![2.png](/static/images/design/UserAvatar/2%201.png)

**自定义尺寸值:** 可选地，头像的尺寸可以设置为自定义值。尽可能使用默认尺寸。

![5.png](/static/images/design/UserAvatar/5.png)
