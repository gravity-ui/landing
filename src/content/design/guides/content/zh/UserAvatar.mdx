用户头像用于快速识别特定用户或账户，并显示已授权用户。

# 尺寸

![1.png](/static/images/design/UserAvatar/1.png)

**XL** - 50 x 50 px

**L** - 42 x 42 px

**M** - 32 x 32 px（默认）

**S** - 28 x 28 px

**XS** - 24 x 24 px

<aside>
💡这些标准头像尺寸确保周围的文本可读。
</aside>

# 内容类型

![2.png](/static/images/design/UserAvatar/2.png)

**无图像（默认）：** 如果图像源缺失或加载错误，将使用默认占位符填充容器。

**上传的图像：** 如果用户上传图像，该图像将用作整个头像区域的占位符。默认情况下，图像填充整个容器宽度（100%）。

# 功能


**点击或悬停时的操作：** 可选地，用户头像可能包含在点击或悬停时显示的附加信息。

![2.png](/static/images/design/UserAvatar/2%201.png)

**自定义尺寸值：** 可选地，头像的大小可以设置为自定义值。尽可能使用默认大小。

![5.png](/static/images/design/UserAvatar/5.png)
