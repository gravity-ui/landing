一个用于显示头像以及用户名和联系信息的组件，其内容可以轻松修改。

# 结构

![Specs3.png](/static/images/design/User/Specs3.png)

1. 用户头像（Avatar组件）
2. 用户名块（可选）
3. 联系信息块（可选）

# 尺寸

![7.png](/static/images/design/User/7.png)

**XL** - 50x50px头像 + 可根据内容宽度调整的文本框架

**L** - 42x42px头像 + 可根据内容宽度调整的文本框架

**M** - 32x32px头像（默认）+ 可根据内容宽度调整的文本框架

**S** - 28x28px头像 + 可根据内容宽度调整的文本框架

**XS** - 24x24px头像 + 可根据内容宽度调整的文本框架

# 内容类型

![11.png](/static/images/design/User/11.png)

**完整块（默认）**

默认情况下，该组件配置为显示最大量的内容，包括头像、姓名部分和联系方式部分。

**不显示用户头像**

可选择禁用用户头像块。

# 功能

![10.png](/static/images/design/User/10.png)

**可选显示的灵活配置**

如有必要，可以禁用任何组件块。
