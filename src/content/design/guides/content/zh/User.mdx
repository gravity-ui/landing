一个用于显示用户头像、用户名和联系信息的组件，其内容可以轻松修改。

# 结构

![Specs3.png](/static/images/design/User/Specs3.png)

1. 用户头像 (Avatar 组件)
2. 用户名区块 (可选)
3. 联系信息区块 (可选)

# 尺寸

![7.png](/static/images/design/User/7.png)

**XL** - 50x50px 头像 + 可根据内容宽度调整的文本框

**L** - 42x42px 头像 + 可根据内容宽度调整的文本框

**M** - 32x32px 头像 (默认) + 可根据内容宽度调整的文本框

**S** - 28x28px 头像 + 可根据内容宽度调整的文本框

**XS** - 24x24px 头像 + 可根据内容宽度调整的文本框

# 内容类型

![11.png](/static/images/design/User/11.png)

**完整区块 (默认)**

默认情况下，该组件配置为显示最多内容，包括头像、姓名部分和联系信息部分。

**不显示用户头像**

可以根据需要禁用用户头像区块。

# 功能

![10.png](/static/images/design/User/10.png)

**可选显示的灵活配置**

任何组件区块都可以根据需要禁用。
