一个用于显示头像以及用户名和联系信息的组件，其内容可以轻松修改。

# 结构

![Specs3.png](/static/images/design/User/Specs3.png)

1.用户头像（头像组件）
2.用户名区块（可选）
3.接触块（可选）

# 大小

![7.png](/static/images/design/User/7.png)

**XL** -50x50px 头像 + 可根据内容宽度调整的文本框架

**L** -42x42px 头像 + 可根据内容宽度调整的文本框架

**M** -32x32px 头像（默认）+ 可以调整到内容宽度的文本框架

**S** -28x28px 头像 + 可根据内容宽度调整的文本框架

**XS** -24x24px 头像 + 可根据内容宽度调整的文本框架

# 内容的类型

![11.png](/static/images/design/User/11.png)

**全区块（默认）**

默认情况下，该组件配置为显示最大数量的内容，包括头像、姓名部分和联系人部分。

**在不显示用户头像的情况下显示**

或者，可以禁用用户头像区块。

# 功能

![10.png](/static/images/design/User/10.png)

**可选显示器的灵活配置**

如有必要，可以禁用任何组件块。
