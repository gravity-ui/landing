ユーザー名と連絡先情報とともにアバターを表示し、コンテンツを簡単に変更できるコンポーネントです。

# 構造

![Specs3.png](/static/images/design/User/Specs3.png)

1. ユーザーアバター (Avatar コンポーネント)
2. ユーザー名ブロック (オプション)
3. 連絡先ブロック (オプション)

# サイズ

![7.png](/static/images/design/User/7.png)

**XL** - 50x50px アバター + コンテンツの幅に合わせて調整可能なテキストフレーム

**L** - 42x42px アバター + コンテンツの幅に合わせて調整可能なテキストフレーム

**M** - 32x32px アバター (デフォルト) + コンテンツの幅に合わせて調整可能なテキストフレーム

**S** - 28x28px アバター + コンテンツの幅に合わせて調整可能なテキストフレーム

**XS** - 24x24px アバター + コンテンツの幅に合わせて調整可能なテキストフレーム

# コンテンツの種類

![11.png](/static/images/design/User/11.png)

**フルブロック (デフォルト)**

デフォルトでは、アバター、名前セクション、連絡先セクションを含む、最大限のコンテンツを表示するようにコンポーネントが設定されています。

**ユーザーアバターなしで表示**

オプションで、ユーザーアバターブロックを無効にすることができます。

# 機能

![10.png](/static/images/design/User/10.png)

**オプション表示の柔軟な設定**

必要に応じて、どのコンポーネントブロックも無効にすることができます。
