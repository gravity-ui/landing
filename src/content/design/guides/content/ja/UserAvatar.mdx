ユーザーアバターは、特定のユーザーまたはアカウントを素早く識別し、認証されたユーザーを表示するために使用されます。

# サイズ

![1.png](/static/images/design/UserAvatar/1.png)

**XL** - 50 x 50 px

**L** - 42 x 42 px

**M** - 32 x 32 px (デフォルト)

**S** - 28 x 28 px

**XS** - 24 x 24 px

<aside>
💡これらの標準的なアバターの寸法は、周囲のテキストが読みやすいことを保証します。
</aside>

# コンテンツの種類

![2.png](/static/images/design/UserAvatar/2.png)

**画像なし (デフォルト):** 画像ソースが見つからない場合や読み込みエラーが発生した場合、デフォルトのプレースホルダーがコンテナを埋めるために使用されます。

**アップロードされた画像:** ユーザーが画像をアップロードした場合、その画像がアバター領域全体を埋めるプレースホルダーとして使用されます。画像はデフォルトでコンテナの幅全体 (100%) を占めます。

# 機能

**クリックまたはホバー時のアクション:** オプションで、ユーザーアバターにはクリックまたはホバー時に表示される追加情報を含めることができます。

![2.png](/static/images/design/UserAvatar/2%201.png)

**カスタムサイズ値:** オプションで、アバターのサイズをカスタム値に設定できます。可能な限りデフォルトサイズを使用してください。

![5.png](/static/images/design/UserAvatar/5.png)
